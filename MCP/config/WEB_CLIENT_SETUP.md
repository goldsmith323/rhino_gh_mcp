# Alternative MCP Client Setup

## Overview

This configuration provides tool definitions for alternative MCP clients that may require different setup approaches.

## Available Tools

The server provides these tools:

### 1. draw_line_rhino
Draw a line in Rhino 3D space between two points

### 2. list_grasshopper_sliders
List all available slider components in Grasshopper

### 3. set_grasshopper_slider  
Change the value of a Grasshopper slider by name

### 4. get_rhino_info
Get information about the current Rhino session

## Setup Requirements

- Rhino 8 with bridge server running
- MCP server running externally
- Proper client configuration

Refer to your specific MCP client documentation for exact integration steps.
